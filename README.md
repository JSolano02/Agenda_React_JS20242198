# Agenda React

Proyecto web de una agenda de contactos creada con React. Permite consultar contactos existentes, registrar nuevos contactos y buscar dentro del listado.

La aplicación está organizada con un componente padre, un formulario para agregar contactos y un listado para mostrarlos. También separa la conexión con el servicio web, el modelo de contacto y las validaciones.

La agenda usa `fetch` para comunicarse con el servicio externo y mantiene la interfaz actualizada desde React.
